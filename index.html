Examples

Create a new instance of intermezzo.js:

var intermezzo = new Intermezzo()

Add an event to the timeline:

var eventId = intermezzo.addEvent(time, duration, callback)

Begin playback:

intermezzo.play()

Pause playback:

intermezzo.pause()

Seek to a time in milliseconds:

intermezzo.seek(time)
